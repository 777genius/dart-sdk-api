dart.typed_data.implementation ​

Specialized integers and floating point numbers, with SIMD support and efficient lists.

NativeFloat32x4Implementation of Dart Float32x4 immutable value type and operations. Float32x4 stores 4 32-bit floating point values in "lanes". The lanes are "x", "y", "z", and "w" respectively.
NativeFloat32x4ListA fixed-length list of Float32x4 numbers that is viewable as a TypedData. For long lists, this implementation will be considerably more space- and time-efficient than the default List implementation.

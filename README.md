# Dart SDK API Documentation

Modern VitePress-based API documentation for the [Dart SDK](https://github.com/dart-lang/sdk).

Generated by [dartdoc-vitepress](https://github.com/777genius/dartdoc-vitepress) — a Dart documentation generator that produces VitePress sites.

## Live Site

**https://777genius.github.io/dart-sdk-api/**

## Features

- Full Dart SDK API reference with linked type signatures
- Full-text search across all libraries
- Dark/light mode
- Responsive mobile layout
- Multi-line Dart-style code formatting

## Libraries

Core libraries: `dart:core`, `dart:async`, `dart:collection`, `dart:convert`, `dart:math`, `dart:typed_data`, `dart:io`, `dart:isolate`, `dart:developer`, `dart:ffi`

Web libraries: `dart:html`, `dart:js_interop`, `dart:ui`, `dart:ui_web`

## Local Development

### Prerequisites

- [Dart SDK](https://dart.dev/get-dart) 3.10+
- [Node.js](https://nodejs.org/) 18+
- [dartdoc-vitepress](https://github.com/777genius/dartdoc-vitepress) cloned locally

### Generate Docs

```bash
dart run ~/dev/projects/dartdoc-vitepress/bin/dartdoc_vitepress.dart \
  --sdk-docs --format vitepress --output .
```

### Dev Server

```bash
npm install
npx vitepress dev --port 5174
```

### Build & Deploy

```bash
# Full pipeline: generate → build → deploy
./deploy.sh

# Or step by step:
./deploy.sh --generate   # Generate API markdown
./deploy.sh --build      # Build VitePress static site
./deploy.sh --deploy     # Deploy to GitHub Pages
```

> **Note:** VitePress build requires ~16-24 GB RAM for 1800+ pages.
> Set `NODE_OPTIONS="--max-old-space-size=24576"` if needed.
